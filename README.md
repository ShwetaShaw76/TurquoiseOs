# TurquoiseOs
- A simulation of a custom web-based Os
- It features an android system
- It has apps and even a boot screen and an app Screen
-  It has a top Panel which has features like brightness and flightMode 


## Unique features
- Brightness and volume can be adjusted by the brightness bar and volume bar 
- Brightness and volume bar are made sliders
- drag function from homescreen to go to topPanel and appScreen
- drag down while clicking the mouse the notification state( the top bar) to get to the topPanel from the homeScreen 
- drag up while clicking the mouse the app Panel in the homeScreen to get to the appScreen

## Featres not Implemented
- Any app won't work as it was not my primary goal to make apps( I just wanted to design Os) 
- The Search Bar at the appScreen is not functional
- any changes in the topPanel will not be shown in the top bar in the homeScreen 
- Only optimized for laptop or computer, in small screen the webpage enlarges
